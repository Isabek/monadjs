# monadjs

## Monad Library for JavaScript

### Install
    $ npm install monadjs

### Usage
#### Nodejs
    var monads = require("monadjs");
#### Browser
Use [browserify](http://browserify.org/) to use in browser.

### Docs

#### Supports  
* Identity Monad
* Maybe Monad
* Array Monad
* State Monad
* Continuation Monad
* Parser Monad

See examples folder.
